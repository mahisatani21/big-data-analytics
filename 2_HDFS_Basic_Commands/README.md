# HDFS Basic Commands

This repository contains a collection of basic **Hadoop Distributed File System (HDFS)** commands used to perform common file operations. It serves as a quick reference for beginners learning how to manage files and directories in HDFS.

## Prerequisites

* Apache Hadoop installed and configured
* HDFS services running
* Access to the Hadoop command line
* Sample text files for testing

## Topics Covered

* Creating directories in HDFS
* Uploading files to HDFS
* Listing files and directories
* Viewing file contents
* Downloading files from HDFS
* Checking the replication factor of files

## File Operations Included

* Create a new directory in HDFS.
* Upload one or more files from the local system to HDFS.
* Display the contents of HDFS directories.
* Read the contents of a file stored in HDFS.
* Download files from HDFS to the local file system.
* Verify the replication factor assigned to HDFS files.

## Repository Contents

* HDFS directory creation commands
* File upload commands
* Directory listing commands
* File viewing commands
* File download commands
* Replication factor verification commands

## Learning Outcomes

After completing this guide, you will be able to:

* Create and manage directories in HDFS.
* Upload files from the local system to HDFS.
* View files stored in HDFS.
* Retrieve files from HDFS.
* Understand and verify HDFS replication.
* Perform basic HDFS file management tasks.

## Notes

* Ensure Hadoop and HDFS services are running before executing the commands.
* Execute the commands in the order provided for a smooth learning experience.
* The examples assume the use of sample text files stored on the local machine.
