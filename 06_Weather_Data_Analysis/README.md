# Weather Data Analysis using Hadoop MapReduce

This repository demonstrates **Weather Data Analysis using Hadoop MapReduce** with **Hadoop Streaming** and custom **Python Mapper and Reducer** programs. The project processes large-scale weather temperature data stored in HDFS to identify the hottest and coldest days efficiently using distributed computing.

## Objective

The objective of this project is to:

* Analyze weather temperature data using Hadoop MapReduce.
* Process large-scale weather datasets with Hadoop Streaming.
* Implement custom Python Mapper and Reducer programs.
* Identify the hottest day based on maximum temperature.
* Identify the coldest day based on minimum temperature.

## Prerequisites

* Ubuntu Linux
* Apache Hadoop installed and configured
* Java (OpenJDK 11 or later)
* Python 3
* Hadoop Streaming JAR
* HDFS and YARN services running

## Workflow

The project follows these steps:

1. Create a weather dataset.
2. Generate a large weather dataset for distributed processing.
3. Develop a Python Mapper program.
4. Develop Python Reducer programs for hottest and coldest day analysis.
5. Upload the dataset to HDFS.
6. Execute Hadoop Streaming MapReduce jobs.
7. View and analyze the generated results.
