# Sales Analysis using Hadoop MapReduce – Top Products by Total Sales

This repository demonstrates **Sales Analysis using Hadoop MapReduce** with **Hadoop Streaming** and custom **Python Mapper and Reducer** programs. The project processes large-scale sales data stored in HDFS to identify the **Top 5 products based on total sales** and perform **city-wise product sales analysis** using distributed computing.

## Objective

The objective of this project is to:

* Analyze sales data using Hadoop MapReduce.
* Process large-scale sales datasets using Hadoop Streaming.
* Implement custom Python Mapper and Reducer programs.
* Calculate total sales for each product.
* Identify the Top 5 products based on total sales.
* Perform city-wise product sales analysis.

## Prerequisites

* Ubuntu Linux
* Apache Hadoop installed and configured
* Java (OpenJDK 11 or later)
* Python 3
* Hadoop Streaming JAR
* HDFS and YARN services running

## Workflow

The project follows these steps:

1. Create a sales dataset.
2. Generate a large sales dataset for distributed processing.
3. Develop a Python Mapper program for product sales.
4. Develop a Python Reducer program to calculate total product sales.
5. Upload the dataset to HDFS.
6. Execute the Hadoop Streaming job.
7. View the generated output.
8. Perform city-wise product sales analysis.
