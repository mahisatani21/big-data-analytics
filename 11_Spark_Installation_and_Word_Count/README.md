# 🚀 Apache Spark Installation & First PySpark Exercise

## 📌 Project Title

**Apache Spark Installation, Configuration, and Word Count using PySpark DataFrames**

---

# 📖 Overview

This project demonstrates the complete installation, configuration, and verification of **Apache Spark 4.1.x** on Ubuntu/Linux. It also includes the implementation of the classic **Word Count** application using **PySpark DataFrames**, providing a foundation for distributed data processing and analytics.

The project is divided into two phases:

- **Phase 1:** Apache Spark Installation & Configuration
- **Phase 2:** Word Count using PySpark DataFrames

---

# 🎯 Objectives

### Phase 1

- Install OpenJDK 21
- Install Apache Spark
- Configure environment variables
- Verify Spark installation
- Launch Spark Shell and PySpark

### Phase 2

- Understand PySpark
- Create distributed datasets
- Work with DataFrames
- Perform Word Count
- Read text files using Spark
- Understand Spark transformations and actions

---

# 🖥️ Software Requirements

| Software | Version |
|----------|----------|
| Ubuntu Linux | 22.04 or Later |
| Java | OpenJDK 21 |
| Apache Spark | 4.1.x |
| Hadoop | 3.x |
| Python | 3.x |

---

# 🔹 Phase 1 – Apache Spark Installation

This phase covers the complete installation and configuration of Apache Spark.

The activities performed include:

- Updating Ubuntu packages
- Installing OpenJDK 21
- Configuring JAVA_HOME
- Downloading Apache Spark
- Extracting Spark binaries
- Moving Spark to the installation directory
- Configuring SPARK_HOME and PATH
- Verifying the installation
- Launching Spark Shell
- Launching PySpark
- Checking the installed Spark version

---

# 🔹 Phase 2 – First PySpark Exercise

This phase demonstrates the implementation of the classic Word Count program using PySpark DataFrames.

The activities performed include:

- Starting PySpark
- Creating a distributed dataset (RDD)
- Converting the RDD into a DataFrame
- Splitting sentences into words
- Counting the frequency of each word
- Sorting the results
- Displaying the final output
- Reading text from an external file
- Performing Word Count on the text file
