# Hadoop Word Count Using MapReduce

This repository demonstrates the implementation of the **Word Count** application using the Hadoop **MapReduce** framework. The experiment covers creating an input text file, uploading it to HDFS, executing the built-in Hadoop Word Count program, and viewing the generated word frequency results.

## Objective

The objective of this project is to:

* Create a text file containing sample data.
* Upload the file to Hadoop Distributed File System (HDFS).
* Execute the Hadoop MapReduce Word Count application.
* Analyze the generated output showing the frequency of each word.

## Prerequisites

* Ubuntu Linux
* Apache Hadoop installed and configured
* HDFS and YARN services running
* Java (OpenJDK 11 or later)
* Hadoop MapReduce Examples JAR

## Workflow

The experiment consists of the following steps:

1. Create the input text file.
2. Verify the contents of the input file.
3. Upload the file to HDFS.
4. Execute the Hadoop Word Count MapReduce program.
5. Check the output directory in HDFS.
6. View the generated word frequency results.

## Repository Contents

* Input text file creation
* HDFS file upload
* Hadoop MapReduce execution
* Output directory verification
* Word count result visualization
* Screenshots of each execution step

## Expected Output

The Hadoop MapReduce program processes the input file and generates an output containing the frequency of each word. The results are stored in the specified HDFS output directory.

## Learning Outcomes

After completing this project, you will be able to:

* Understand the Hadoop MapReduce execution workflow.
* Store input data in HDFS.
* Execute built-in Hadoop MapReduce applications.
* View and interpret MapReduce output.
* Understand how distributed processing is performed using Hadoop.
